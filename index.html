Programa Gestión - Universidad

Objetivo:

El objetivo del programa es poder trabajar con la base de datos de una universidad. Consta de un menú principal repartido en cuatro partes: alumnos, profesores, matriculaciones y asignaturas. Las funciones que se podrán llevar a cabo serán: dar de alta, dar de baja, modificar y hacer consultas sobre la base de datos.
